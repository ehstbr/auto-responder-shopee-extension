# 🤝 Contribuindo

Obrigado pelo interesse em contribuir com o **Auto Responder Shopee**.

## Como contribuir

Você pode ajudar com:

- correções de bugs;
- melhorias de interface;
- ajustes de compatibilidade com mudanças da Shopee;
- documentação;
- revisão de segurança e privacidade;
- sugestões de novas funcionalidades.

## Antes de abrir uma issue

Verifique:

1. Se você está usando a versão mais recente.
2. Se recarregou a extensão em `chrome://extensions/`.
3. Se atualizou a página da Shopee após recarregar a extensão.
4. Se o problema acontece também em modo teste.

## Boas práticas

Ao relatar bugs, inclua:

- versão da extensão;
- navegador usado;
- passo a passo para reproduzir;
- print ou vídeo, se possível;
- mensagem de erro do console, se houver.

Não envie dados sensíveis, como informações de clientes, pedidos, vendas ou credenciais.
