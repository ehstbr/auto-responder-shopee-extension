# 🔒 Política de Privacidade

Última atualização: julho de 2026

O **Auto Responder Shopee** foi criado com foco em transparência e uso local no navegador.

## 1. Dados que a extensão não coleta

A extensão não foi projetada para coletar, armazenar ou enviar para servidores da EduhCommerce:

- dados de clientes;
- dados de pedidos;
- dados de vendas;
- texto de avaliações;
- texto das respostas cadastradas;
- dados financeiros;
- credenciais, senhas ou tokens;
- informações privadas da conta Shopee.

## 2. Dados salvos localmente

A extensão salva localmente no navegador algumas configurações necessárias para funcionamento, como:

- respostas cadastradas por nota;
- modo teste ativado/desativado;
- limites de execução;
- tempos de espera configurados;
- aceite dos termos de uso;
- última contagem conhecida de avaliações para responder.

Esses dados são salvos usando `chrome.storage.local` e ficam no próprio navegador do usuário.

## 3. Comunicação com a Shopee

Para funcionar dentro do Seller Center, a extensão interage com a página da Shopee carregada no navegador.

Em alguns casos, a extensão consulta endpoints internos da própria Shopee, dentro da sessão autenticada do usuário, para obter informações como contagens de avaliações a responder.

Essas consultas são feitas diretamente entre o navegador do usuário e a Shopee. A EduhCommerce não recebe esses dados.

## 4. Comunicação com servidores da EduhCommerce

A versão atual não envia dados para servidores da EduhCommerce.

Caso versões futuras implementem analytics, licenciamento, cadastro, suporte integrado ou outro recurso com envio de dados, esta política deverá ser atualizada e a coleta deverá ser informada de forma clara ao usuário.

## 5. Dados de contato

O usuário pode entrar em contato voluntariamente por canais públicos do projeto, como GitHub Issues, redes sociais ou e-mail ofuscado informado no README.

Dados enviados voluntariamente nesses canais serão tratados apenas para fins de suporte, comunicação e melhoria do projeto.

## 6. Segurança

Mesmo com o cuidado no desenvolvimento, nenhuma ferramenta é totalmente imune a falhas. Use a extensão com atenção, principalmente em modo de envio real.

## 7. Alterações nesta política

Esta política pode ser atualizada conforme o projeto evoluir.
