# 🔐 Segurança

Se encontrar uma vulnerabilidade ou comportamento que possa expor dados, evite publicar detalhes sensíveis diretamente em uma issue pública.

## Como reportar

Preferencialmente, use os recursos de segurança do próprio GitHub, como **Private vulnerability reporting**, se estiver habilitado no repositório.

Alternativamente, entre em contato pelo e-mail ofuscado:

```txt
contato [arroba] eduhcommerce.com.br
```

## Escopo

São considerados relevantes:

- vazamento de dados;
- coleta não intencional de informações;
- comportamento inesperado em páginas da Shopee;
- falhas que possam causar ações indevidas em massa.
