# 🛒 Publicação na Chrome Web Store

Este documento serve como checklist interno para publicação.

## Antes de publicar

- [ ] Revisar `manifest.json`.
- [ ] Confirmar versão.
- [ ] Testar instalação limpa.
- [ ] Testar modo teste.
- [ ] Testar envio real com limite baixo.
- [ ] Revisar política de privacidade.
- [ ] Revisar termos de uso.
- [ ] Gerar ZIP com `scripts/package-extension.sh`.

## Arquivos úteis

- `PRIVACY.md`
- `TERMS.md`
- `store-assets/descriptions/descricao-curta.txt`
- `store-assets/descriptions/descricao-completa.txt`
- `store-assets/descriptions/justificativa-permissoes.txt`
