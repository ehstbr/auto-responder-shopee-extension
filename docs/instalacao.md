# 🚀 Instalação manual

## 1. Baixe o projeto

Você pode baixar o ZIP do GitHub ou clonar o repositório:

```bash
git clone https://github.com/ehstbr/auto-responder-shopee-extension.git
```

## 2. Abra a página de extensões do Chrome

Acesse:

```txt
chrome://extensions/
```

## 3. Ative o modo desenvolvedor

Ative a opção **Modo do desenvolvedor**.

## 4. Carregue a extensão

Clique em **Carregar sem compactação** e selecione a pasta:

```txt
extension/
```

## 5. Use no Seller Center

Acesse:

```txt
https://seller.shopee.com.br/
```

Na página de avaliações, clique no botão **🤖 Auto Responder**.
